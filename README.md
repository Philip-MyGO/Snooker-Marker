# Snooker-Marker
A simple score tracker for snooker using Swift that it can:
- Track Scores Separately: Keep accurate live scores for two players side by side.
- Switch Active Player: Change who is currently hitting with a simple tap.
- Custom Names: Type in and change player names directly on the screen.
- Input Points Quickly: Tap colored ball icons to add standard points (1 to 7).
- Handle Fouls: Choose a penalty value (4 to 7 points) and automatically give it to the opponent.
- Declare Free Balls: Choose any color ball to count as 1 point when a free ball is awarded.
- View Margin Deficits: See exactly how many points a player is leading or behind in real time.
- View Remaining Points: Watch the maximum remaining table points count down accurately, including special logic for the final red and transition color.
- Undo Mistakes: Tap to reverse any action step-by-step, including scores, fouls, free balls, and resets.
- Clear Table: Reset the entire match back to 147 points with an option to undo the reset if tapped by accident.
